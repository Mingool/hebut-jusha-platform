## 聚沙公益平台
名字是我实验室一个学姐起的...

> 所选框架

jquery-mobile Struts2 Hibernate mysql5

> 流程

注册登录 -> 寻找捐赠需求 -> 预约捐赠并获得捐赠码 -> 按时去指定地点捐赠 -> 完成捐赠 -> 获得积分
